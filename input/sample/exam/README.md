# Sample Exam

This directory should contain a sample exam paper PDF (e.g., `sample_exam.pdf`).

## Sample Exam Structure

A typical exam paper should include:

### Section A (40 marks)
1. **Question 1 (10 marks)**
   a) Define a matrix and explain its properties. [4 marks]
   b) Calculate the determinant of a 3x3 matrix. [6 marks]

2. **Question 2 (15 marks)**
   a) Explain the concept of a derivative. [5 marks]
   b) Apply the chain rule to differentiate a composite function. [10 marks]

### Section B (30 marks)
3. **Question 3 (15 marks)**
   a) Define a random variable. [5 marks]
   b) Calculate the expected value of a discrete distribution. [10 marks]

4. **Question 4 (15 marks)**
   a) Explain Big-O notation. [7 marks]
   b) Analyze the time complexity of a sorting algorithm. [8 marks]

**Note:** Create a PDF with similar content for testing purposes.
